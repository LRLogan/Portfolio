# Portfolio Command Deck

A Vite vanilla JavaScript portfolio scaffold with three primary views:

- Intro
- Projects
- Contact / Links

The project page includes expandable project detail sections so each project can grow from a compact showcase into a deeper case study.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

The local dev server is configured for `http://127.0.0.1:5173/` by default.
